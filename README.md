# Odin Recipes

## Description
This project is part of The Odin Project curriculum. It is a basic recipe website built to practice fundamental HTML skills.

## Skills Demonstrated
* Creating and navigating directory structures.
* Using Boilerplate HTML.
* Working with headings, paragraphs, and lists.
* Implementing relative and absolute links.
* Adding and configuring images.
* Version control with Git and GitHub.
